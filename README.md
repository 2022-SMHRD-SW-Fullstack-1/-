# 디지몬야구게임
팀명 : 맨앞조
## 1. 기획
데이터베이스와 java를 연결해 활용해서 미니 게임 만들기
## 2. 주요기능
 * 랜덤으로 대상 뽑은 후 배틀
 * 아스키 코드를 이용한 시각화 기능
 * 백그라운드 사운드를 사용
 * 경기 결과 저장하는 랭킹페이지
 * 회원 관리 기능(회원가입/로그인/닉네임변경/탈퇴)
## 3. 개발환경
* 사용언어 : java 1.8
* 데이터베이스: oracle 11g
